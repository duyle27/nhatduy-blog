Node.js cho phép tạo server HTTP mà không cần phần mềm trung gian.

HTTP Server trong Node.js:
- Nhận request từ client
- Xử lý logic
- Trả response cho client

Đây là nền tảng để xây dựng các framework như Express hoặc NestJS.
