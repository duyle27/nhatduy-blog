Xác thực người dùng là phần quan trọng trong hệ thống web.

- Session: server lưu trạng thái đăng nhập
- JWT: token được gửi kèm mỗi request

JWT phù hợp với REST API và mobile app, trong khi Session thường dùng cho web truyền thống.
