Từ Java 11, Java cung cấp HttpClient để gửi HTTP request một cách đơn giản.

HttpClient cho phép:
- Gửi request GET, POST
- Nhận response dạng String hoặc JSON
- Giao tiếp với REST API

Công cụ này thường được dùng trong các ứng dụng backend Java hoặc khi tích hợp hệ thống.
