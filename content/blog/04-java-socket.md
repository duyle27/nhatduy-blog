Socket TCP cho phép hai chương trình giao tiếp trực tiếp qua mạng.

Trong Java:
- Server dùng ServerSocket để lắng nghe kết nối
- Client dùng Socket để kết nối tới server

Lập trình socket giúp hiểu rõ cách dữ liệu truyền qua mạng ở mức thấp hơn HTTP.
