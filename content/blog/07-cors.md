CORS (Cross-Origin Resource Sharing) là cơ chế bảo mật của trình duyệt.

CORS ngăn website gọi API từ domain khác nếu server không cho phép.  
Điều này giúp hạn chế các tấn công không mong muốn.

Hiểu CORS giúp lập trình viên xử lý lỗi gọi API hiệu quả hơn.
