REST API là cách xây dựng dịch vụ web dựa trên giao thức HTTP và các tài nguyên (resource).

REST sử dụng các HTTP method để thao tác dữ liệu:
- GET: lấy dữ liệu
- POST: thêm dữ liệu
- PUT: cập nhật dữ liệu
- DELETE: xoá dữ liệu

REST API giúp frontend và backend giao tiếp với nhau rõ ràng, dễ mở rộng và phù hợp cho web, mobile app.
