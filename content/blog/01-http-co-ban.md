## HTTP là gì?
HTTP (HyperText Transfer Protocol) là giao thức nền tảng của Web, hoạt động theo mô hình **Client – Server**.

## Thành phần của HTTP
- Request (Client → Server)
- Response (Server → Client)
- Method: GET, POST, PUT, DELETE
- Status code: 200, 404, 500...

## Ví dụ
Trình duyệt gửi request GET tới server để lấy trang web.

## Kết luận
Hiểu HTTP là nền tảng quan trọng cho lập trình Web và API.
