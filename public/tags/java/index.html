Gọi API bằng Java HttpClient

Từ Java 11, Java cung cấp HttpClient để gửi HTTP request một cách đơn giản. HttpClient cho phép: Gửi request GET, POST Nhận response dạng String hoặc JSON Giao tiếp với REST API Công cụ này thường được dùng trong các ứng dụng backend Java hoặc khi tích hợp hệ thống.

Lập trình Socket TCP cơ bản với Java

Socket TCP cho phép hai chương trình giao tiếp trực tiếp qua mạng. Trong Java: Server dùng ServerSocket để lắng nghe kết nối Client dùng Socket để kết nối tới server Lập trình socket giúp hiểu rõ cách dữ liệu truyền qua mạng ở mức thấp hơn HTTP.
